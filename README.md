# 🚀 JobHunter — Personal Job Application Tool

> A fully-featured personal job hunting dashboard built for **Naman Kudesia** — single HTML file, works offline, no installation needed.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏠 **Dashboard** | Daily goal tracker, stats cards, weekly progress, recent activity |
| 💼 **Job Search** | 14+ matched jobs with % match score, filter by role/location/type |
| 📋 **Application Tracker** | Track every application — status, notes, follow-ups |
| ✉️ **HR Email Composer** | 4 personalised templates — cold outreach, follow-up, referral, thank-you |
| 📄 **Resume Vault** | All 4 tailored resumes with "when to use" guide |
| 📊 **Analytics** | Charts for status breakdown, role types, company targets, smart tips |
| ⚙️ **Settings** | Profile config, daily goal, export data as JSON |

---

## 🚀 How to Use

```bash
# Option 1 — Download and open directly
# Download JobHunter.html → double-click → opens in Chrome/Edge

# Option 2 — Clone and open
git clone https://github.com/namankudesia/jobhunter-personal-tool.git
cd jobhunter-personal-tool
open JobHunter.html   # Mac
start JobHunter.html  # Windows
```

**No server needed. No npm install. No dependencies. Just open the file.**

---

## 💾 How data is saved

All your applications, settings, and email logs are saved automatically in your browser's **localStorage**. Your data stays on your device — nothing is sent anywhere.

To back up your data: Settings → Export JSON

## 📬 HR Email Templates included

1. **Cold outreach** — first contact with HR, introduce yourself
2. **Follow-up** — sent 5–7 days after applying online
3. **Referral request** — ask a LinkedIn connection for internal referral
4. **Thank you after interview** — send within 24 hrs to stand out

Each template auto-fills your name, company, role, and HR details.
